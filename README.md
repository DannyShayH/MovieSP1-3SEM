# Movie SP1 Task

- To Run the program, go to TMDB's Website: https://www.themoviedb.org/settings/api
- To get your API Key, log in/sign up. Go to settings tab and on the left go to API
- Copy API Key at the bottom of the screen
  
### On Intellij
- On the program in Intellij
- Click the arrow by Main beside the run button
- Click edit configuration
- Click the paper sheet icon on the right side of **Environment Variables**
- add the name **API_KEY** and the Api Key copied from Tmdb
- Now run the code
